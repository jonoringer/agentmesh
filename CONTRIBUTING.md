# Contributing

Thanks for helping shape AgentMesh.

## Current status

This repository is in its foundation stage. The immediate goal is to harden the vocabulary, specs, and architecture before locking in implementation choices.

## Good first contributions

- improve spec clarity
- challenge ambiguous naming
- add examples from real-world agent systems
- propose API shape improvements
- suggest operator-facing observability requirements

## Development philosophy

- prefer explicit primitives over magical abstractions
- keep the control plane framework-agnostic
- design for operability before novelty

## Before opening large PRs

Please open an issue or discussion for:

- new core primitives
- major API changes
- control plane architecture shifts
- hosted product assumptions
